# IPO Listing Gain Prediction

## Overview
Machine learning model to predict IPO listing gains using historical Indian IPO data. The project focuses on identifying key financial and market factors that influence listing performance.

## Objectives
- Analyze IPO data using exploratory data analysis (EDA)
- Build a predictive model for listing gains
- Evaluate model performance using standard metrics

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Approach
- Data preprocessing and cleaning
- Feature engineering and scaling
- Logistic regression-based classification model
- Model evaluation using accuracy, precision, and recall

## Results
- Accuracy: **67.2%**
- Precision: **70.6%**
- Recall: **68.6%**

## Key Insights
- Certain financial indicators and market conditions significantly influence IPO listing gains
- Model provides balanced performance across precision and recall
- Useful for understanding risk-return behavior in IPO investments
